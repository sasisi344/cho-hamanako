# ⚖️ 浜名湖における釣りに関する法律・条例

このドキュメントは、浜名湖およびその周辺水域で釣り記事を執筆・編集する際の<strong>法律・条例・漁業権・漁期・採捕制限などのリファレンス</strong>です。記事執筆時には本ファイルの内容を**一次情報として参照**し、既存記事との齟齬・古い情報の上書きを防ぎます。

> [!IMPORTANT]
> 本書は<strong>エージェント／執筆者向けの内部リファレンス</strong>です。読者向けの法令・マナー解説記事は [hamanako-fishing-rules-and-manners](../src/content/blog/guide/beginner/hamanako-fishing-rules-and-manners/index.mdx) を参照・更新してください。法令・公示は変更されることがあるため、出典・改訂日を記事内で明記し、不確実な点は読者に「最新の公示・現地看板を確認してください」と促すこと。

## 🗺️ 海域区分（管轄の違い）

浜名湖の釣りルールを語るうえで最も重要なのが、<strong>今切口の大橋（浜名バイパス／浜名大橋）を境とした海域区分</strong>です。

| 海域 | 範囲 | 管轄・適用ルール |
| :--- | :--- | :--- |
| <strong>湖内（北側）</strong> | 今切口の大橋より<strong>北側</strong>の水域（浜名湖本体） | <strong>浜名湖の漁業ルール</strong>（浜名漁業協同組合・静岡県の遊漁規則）が適用 |
| <strong>湖外（南側）</strong> | 今切口の大橋より<strong>南側</strong>（太平洋・遠州灘） | <strong>共同漁業権</strong>の指定水域。沿岸の漁業協同組合の漁業権が優先 |

> [!CAUTION]
> 「浜名湖には共同漁業権が設定されている」という大雑把な書き方は<strong>不正確</strong>になりがちです。共同漁業権の指定水域は<strong>大橋以南の太平洋・遠州灘側</strong>であり、湖内は浜名湖の漁業ルールに従う旨を分けて書いてください。

加えて、今切口周辺は<strong>釣り禁止エリア</strong>（今切口から左右・沖合1,400m、導流堤上、浜名バイパス以南など）が法的に設定されています。詳細は [hamanako-fishing-rules-and-manners](../src/content/blog/guide/beginner/hamanako-fishing-rules-and-manners/index.mdx) を参照。

## 📜 漁業権の基礎（3種類）

漁業権は日本の沿岸水域にほぼ全域で設定されている法的な権利で、執筆時には「どの種類の漁業権の話か」を意識する必要があります。

| 種類 | 内容 | 浜名湖周辺での主な実態 |
| :--- | :--- | :--- |
| <strong>共同漁業権</strong> | 一定地域の漁民が一定の水面を共同利用し、定められた漁業を営む権利 | 浜名湖内（浜名漁協）・遠州灘沿岸の各漁協に免許 |
| <strong>区画漁業権</strong> | 養殖のため、いけす・のり網・かき棚等の施設を設置する権利 | 浜名湖の<strong>のり・かき養殖</strong>が代表 |
| 定置漁業権 | 海面に漁具を定置して魚類を採捕する権利 | 主に伊豆半島沿岸・沼津・由比・焼津側 |

> [!IMPORTANT]
> 釣り記事で「漁業権」と書くときは、文脈上ほとんどが<strong>共同漁業権</strong>を指す。読者が混乱しないよう、必要に応じて「共同漁業権の対象種」と明示する。

### 浜名湖（湖内）の共同漁業権

浜名湖の湖内には<strong>共第20号</strong>として浜名漁業協同組合の共同漁業権が設定されています。

| 免許番号 | 漁業権者 | 漁場の位置 | 漁業権対象種 |
| :--- | :--- | :--- | :--- |
| <strong>共第20号</strong> | 浜名漁業協同組合 | 浜名湖内 | <strong>あさり、はまぐり、なまこ、すじあおのり</strong> |

これら4種を一般遊漁者が採ると<strong>漁業権侵害</strong>として罰せられる可能性があります（罰則は後述）。

### 浜名湖（湖外・遠州灘側）の参考

湖外（今切口大橋以南）は遠州灘沿岸の漁協が共同漁業権を持ち、<strong>いせえび・たこ・あわび・さざえ・なまこ・各種海藻類</strong>などが対象に含まれる漁業権番号が複数設定されています。詳細は[海面における遊漁のルールについて｜静岡県公式ホームページ](https://www.pref.shizuoka.jp/sangyoshigoto/suisan/suisanshigen/1047550/1028113.html)を参照のこと。

## 🚫 全国共通の禁止行為（法律レベル）

下記は釣り場の看板の有無に関わらず、<strong>日本全国どこでも適用</strong>される禁止行為です。記事内で「漁業権」と書く前に、まずこの法律レベルの禁止が大前提にあることを意識してください。

| 内容 | 根拠条文 |
| :--- | :--- |
| <strong>あわび・なまこ・うなぎ稚魚（しらすうなぎ）</strong>の採捕禁止 | 漁業法 第132条（特定水産動植物の採捕の禁止） |
| 爆発物を使った水産動植物の採捕禁止 | 水産資源保護法 第5条 |
| 有毒物を使った水産動植物の採捕禁止 | 水産資源保護法 第6条 |
| 水産動植物に有害な物の遺棄・漏えい禁止 | 静岡県漁業調整規則 第44条 |
| 保護水面区域での水産動植物の採捕禁止 | 静岡県漁業調整規則 第34条 |
| 潜水器（アクアラング等・簡易潜水器含む）を使った水産動植物の採捕禁止 | 静岡県漁業調整規則 第37条 |

> [!CAUTION]
> <strong>あわび・なまこ・うなぎ稚魚</strong>の3種は「特定水産動植物」として罰則がとくに重く、漁業権侵害の枠を超えた重罰対象（最大3年以下の懲役または3000万円以下の罰金）になります。

## 🐙 リリース基準のガイドライン

漁期内であっても、<strong>資源保護の観点から小型個体はリリース</strong>することを記事内で推奨してください。

### マダコ

- <strong>基準</strong>：頭部（外套膜）が<strong>5cm以下</strong>、または「ピンポン玉大」と表現されるサイズ
- <strong>理由</strong>：
  - 小型個体はアグレッシブで釣れやすいが、<strong>可食部が少ない</strong>
  - 産卵を経験していない若い個体を逃がすことで<strong>翌年以降の資源</strong>に繋がる
  - 倫理的・持続可能性の観点からも逃がすのが正解
- <strong>記事内での書き方の例</strong>：
  > 頭部5cm以下（ピンポン玉大）の小さな個体はリリースを推奨します。小型はアグレッシブで釣りやすい反面、可食部が少ないため、未来の資源を残す意味でも逃がすのが倫理的な選択です。

### 他魚種を追加するときの方針

将来、シーバス・クロダイ・キス等で同様の基準が判明した場合は、<strong>「魚種」「基準（cm/g）」「理由」「出典」</strong>の4点セットで本表に追記すること。

## 🚫 禁止漁具・漁法（再掲）

| 漁具・漁法 | 規制内容 |
| :--- | :--- |
| 潜水器（アクアラング）・簡易潜水器を使った水産動植物の採捕 | 禁止（懲役または高額罰金） |
| 水中眼鏡を使ったやす漁 | 禁止 |
| 幅15cmを超えるクマデ | 禁止（1万円未満の科料） |

### 夜間の灯火採捕（たきや漁・エビすくい漁）

たきや漁・エビすくい漁は浜名湖の文化として有名ですが、これらは<strong>「夜間に灯火で銛・網を使った漁を認められた業者だけが行える特殊な漁」</strong>です。

- 一般遊漁者が夜間に水面を照らして魚介を採取する行為は<strong>無許可の灯火採捕</strong>として違反に問われる可能性
- 特に<strong>シラスウナギ採取期間（晩秋〜冬）</strong>はパトロールが強化される
- 夜釣り中に「ライトで水面を照らしながら銛を構える」ような行為は誤解を招くため避ける

> [!TIP]
> 夜釣り・夜間ロックフィッシュ・夜のチニング記事を書くときは、上記の文化（たきや漁・エビすくい）に簡単に触れつつ、<strong>「業者専用の漁であり一般は不可」</strong>と明示しておくと安全。

## 🛂 河川と海の管轄の違い

漁業権は海面と河川面で管轄が分かれます。河川は河川事務として淡水管理になり、流域ごとの<strong>遊漁規則</strong>が別に設定されています。

| エリア | 漁業権・遊漁券の扱い | 記事執筆時の注意 |
| :--- | :--- | :--- |
| 浜名湖の堤防・砂浜・湖内 | 海面の漁業権＋静岡県遊漁規則。<strong>遊漁券は原則不要</strong> | 湖内固有のサイズ・期間制限のみ要注意 |
| 都田川（河口・汽水部） | <strong>ハゼ・テナガエビは漁業権対象外</strong>で気にしなくてよい | ハゼ釣り記事でも特別注記は不要 |
| 都田川（上流） | <strong>アユは漁業権対象</strong>で<strong>遊漁券が必要</strong> | 都田川アユ記事は必ず遊漁券に言及 |
| 天竜川流域 | <strong>テナガエビ等が漁業権対象</strong>。<strong>禁漁区・禁漁期間あり</strong> | 天竜川絡みの記事では各漁協公示を必ず確認 |

### 遊漁券（入漁券）

- 浜名湖の堤防・砂浜からの一般的な海釣り：<strong>原則不要</strong>
- 都田川など流入河川の上流：<strong>各漁協の遊漁券が必要</strong>な場合がある
- 河川記事を書く際は、必ず該当漁協の最新公示を確認すること

## ⚖️ 罰則の整理

執筆時に「違反すると何が起こるか」を読者に提示する場面では、以下の3段階を意識して使い分けます。

| 違反内容 | 罰則の目安 | 根拠 |
| :--- | :--- | :--- |
| 通常の<strong>漁業権侵害</strong>（共第20号のアサリ・ハマグリ・ナマコ等を採捕） | <strong>100万円以下の罰金</strong> | 漁業法 |
| <strong>特定水産動植物</strong>（あわび・なまこ・うなぎ稚魚）の無許可採捕 | <strong>3年以下の懲役 または 3000万円以下の罰金</strong> | 漁業法 第132条 |
| 今切口の釣り禁止エリアへの立入・釣行 | 6ヶ月以下の懲役 または 10万円以下の罰金 | 港則法等 |

> [!CAUTION]
> 「漁業権侵害は10万円〜の罰金」のような曖昧な表現は<strong>過小評価</strong>になりやすい。執筆時は<strong>「100万円以下」「特定水産動植物なら最大3000万円・懲役3年」</strong>を原則として書き、必要なら「実際の量刑は事案による」と添える。

### 摘発の現実（現行犯主義と「採捕の意思」）

- 漁業権侵害・採捕禁止違反は基本的に<strong>現行犯</strong>での摘発が前提
- ただしSNS時代では、違反行為の写真・動画が事後拡散され、事実上の通報・証拠提出に繋がる事例も増加
- 「砂を掘っていただけ」と言い訳しても、<strong>バケツ・クマデ・網袋などを所持</strong>していれば<strong>採捕の意思</strong>があったと判断されやすい
- 記事内では「現行犯主義だから大丈夫」のような甘い書き方をせず、<strong>「知らなかったでは済まない」</strong>のトーンで統一する

## ✍️ 記事への落とし込みルール

1. <strong>境界の明示</strong>：共同漁業権・採捕制限を語るときは、必ず「今切口の大橋から南／北」のどちらの話かを書き分ける。
2. <strong>漁期の明示</strong>：対象魚種の漁期を記事内 Callout・WARNING・IMPORTANT のいずれかで明示する。シーズナルカレンダー（`CatchChart` 等）も漁期と整合させる。
3. <strong>リリース基準の明示</strong>：体長・体重ではなく、本書に倣って<strong>具体的な観測しやすい基準</strong>（例：頭部5cm／ピンポン玉大）で書く。
4. <strong>罰則の明示</strong>：違反行為に触れる場合は「漁業権侵害＝100万円以下」「特定水産動植物＝最大3000万円・懲役3年」を原則とし、過小評価しない。
5. <strong>出典の明示</strong>：法律・条例に触れる場合は<strong>静岡県公式ホームページ</strong>もしくは<strong>漁業法・水産資源保護法・静岡県漁業調整規則</strong>の条文番号まで添える。
6. <strong>注意喚起</strong>：法令・公示は改定されるため、記事末尾に「現地看板・漁協公式情報の最新確認」を促す一文を必ず添える。
7. <strong>誤情報の上書き</strong>：旧記事に「100g以下リリース」「浜名湖全域に共同漁業権」「漁業権侵害は10万円程度」など本書と矛盾する記述があれば、本書を一次情報として優先し、旧表現を修正する。
8. <strong>夜釣り記事の注意</strong>：たきや漁・エビすくい漁は<strong>業者専用</strong>であり、一般遊漁者の夜間灯火採捕は違反になり得る旨を、夜釣り系記事で必要に応じ明示。
9. <strong>河川記事の注意</strong>：浜名湖流入河川（都田川・天竜川等）を扱うときは、河川独自の遊漁規則・漁業権対象種を漁協公示で確認したうえで遊漁券の要否に言及する。
