# 📱 SNS Promotion Workflow (Friday Launch)

## 📁 出力先
- **ディレクトリ**: `.workspace/_sns-post/weeklymagazine/YYYY/MM/WW`
- **デフォルトファイル名**: `MMDD-sns.md` (例: 0402-sns.md)

このワークフローは、木曜夜に完成した「週刊釣果レポート」の内容を元に、週末の釣行準備を始めるアングラーに向けたSNS（X / Instagram）用の投稿テキストを生成します。

## 🎯 目的
- 浜名湖アングラーに「今週末、どこで何を狙えばいいか」を端的に伝える。
- 釣！浜名湖の最新レポートへの誘導。
- 「私（さしし）」のリアルな声を届けて親近感と信頼性を高める。

## 📅 タイミング
- **作成**: 木曜夜（レポート完成直後）
- **投稿推奨**: 金曜朝 07:00 〜 08:30 または 金曜夜 19:00 〜 21:00

## 📝 入力データ
- `.data-set/research/week-report/YYYY/MM/W[n]/index.md` の内容

---

## 🛠️ 出力フォーマット

### 1. X (Twitter) 用：速報・期待値重視
**文字数**: 140文字以内（+ハッシュタグ）
**構成**:
- 【フック】今週末、最も熱いエリアor魚種を1文で。
- 【詳細】レポートから抜粋した「ベイトの動き」や「特筆すべきパターン」。
- 【さししの独り言】「私ならここ狙うかな...」という個人的な予測。
- 【誘導】記事へのリンク。
  - **URL形式**: `https://cho-hamanako.info/blog/[スラッグ]/`
- **ハッシュタグ**: #浜名湖釣り #浜名湖チニング #奥浜名湖 #表浜名湖 #釣浜名湖

### 2. Instagram 用：まとめ・ガイド重視
**構成**:
- **1枚目想定テキスト**: 「今週末の浜名湖攻略ガイド（W[n]）」
- **本文**:
  - 🔸 今週の総括（1週間の釣果状況）
  - 🔹 週末の特注エリア（奥・中・表から1箇所）
  - 🔹 おすすめルアー / パターン
  - 🎣 さししのワンポイントアドバイス
- **ハッシュタグ**: #浜名湖 #浜名湖釣り #シーバス #クロダイ #キビレ #浜名湖フィッシング #フィッシングライフ

---

## 🤖 エージェントへの指示
1. 週刊レポートを読み込み、最も「アングラーが喜ぶ情報（例：バチ抜け開始、サッパ接岸等）」を特定する。
2. 熟練アングラー「私（さしし）」の口調（です・ます、親しみやすさ、専門性）でテキストを構成する。
3. 絵文字を適度に使用し、画面上での視認性を高める。
4. **【重要】**: 捏造した情報を入れない。レポートにある事実のみをベースに「予測」を添える。
