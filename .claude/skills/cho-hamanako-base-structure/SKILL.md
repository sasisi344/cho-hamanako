# Cho! Hamanako — 基本記事構成

**正本**: `.agents/base-article-structure/SKILL.md`

## 深度（下限）

| 種別 | 本文下限 |
|------|----------|
| 通常（guide / season / travel / theory） | **2,000字** |
| ポイント単体ピラー | **3,000字** |
| まとめ・複数ポイント | 2,000字（各ポイントは短く差別化） |

## モジュール（抜粋）

`M0` リード → `M1` 結論 → `M3` 核心 → `M4` 実践 → `M7` 安全 → `M8` まとめ → `M9` 回遊（BlogCard）

タイプ別の組み合わせは正本の表を参照。

## ポイント複数紹介

執筆前に `.agents/point-article/point-comparison-matrix.md` で比較表と H2 グルーピングを作成する。

## 温度

同カテゴリの完成記事を1本読み、冒頭・語尾・1文1段落・`<strong>` を揃える。詳細は正本「温度の取り方」。
