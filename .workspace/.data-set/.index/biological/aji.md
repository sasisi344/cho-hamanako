# 浜名湖リサーチデータ：マアジ（夏〜秋）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「回遊・成長期」**。
- **活性:** 6月〜10月。黒潮に乗って今切口から湖内へ入ってくる群れを狙う。
- **特徴:** 浜名湖のアジは「豆アジ」から「小アジ」サイズがメイン。数釣りが楽しめる。

## 2. 適水温と魚の居場所の推移
- **適水温:** 20℃〜25℃。
- **居場所:** 潮通しの良い堤防、澪筋付近、常夜灯周辺。
- **主要ポイント:** 新居海釣り公園、舞阪堤防、網干場、砂揚げ場。

## 3. メインベイト（捕食対象）と集まる条件
- **動物性プランクトン:** 潮の流れに乗って回遊。
- **アミエビ:** サビキ釣りで撒かれるコマセ。
- **集まる条件:** 潮通しが良く、かつプランクトンが豊富なエリア。

## 4. 最も有効なアプローチ
- **サビキ釣り:** 5号〜10号のオモリを使用。朝夕のマヅメ時が最大のチャンス。
- **アジング:** 夜間の常夜灯周りでジグヘッド＋ワーム。
- **コツ:** 群れを留めるために「コマセ（アミエビ）」を切らさないことが肝要。
