# 浜名湖リサーチデータ：アオリイカ（春・秋）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「産卵接岸・成長期」**。
- **活性:** 春（4月〜6月）は親イカの大型。秋（9月〜11月）は新子の数釣り。
- **特徴:** 浜名湖内でも年々釣果が安定。藻場が狙い目。

## 2. 適水温と魚の居場所の推移
- **適水温:** 16℃〜25℃（18℃以上で活性化）。
- **居場所:** 潮通しの良い場所のアマモ帯、カキ殻周り、障害物付近。
- **主要ポイント:** 新居海釣り公園、舞阪堤防、砂揚げ場、乙女園周辺。

## 3. メインベイト（捕食対象）と集まる条件
- **小魚:** アジ、イワシ、キス。
- **エビ:** テッポウエビ等。
- **集まる条件:** 潮通しが良く、産卵に適した藻場（アマモ）や隠れ家。

## 4. 最も有効なアプローチ
- **エギング:** 2.5号〜3.5号のエギ。春は大型狙いで3.5号。
- **コツ:** 浜名湖の速い潮では、潮止まり前後か反転流を狙うのが定石。
