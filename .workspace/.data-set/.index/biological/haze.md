# 浜名湖リサーチデータ：ハゼ（夏〜秋）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「成長・落ち期」**。
- **活性:** 8月〜11月。夏場は浅場で数釣り、秋はサイズアップした「落ちハゼ」が深場へ向かう。
- **特徴:** 「浜名湖の風物詩」。一年中釣果が望めるが、秋が最も面白い。

## 2. 適水温と魚の居場所の推移
- **適水温:** 18℃〜25℃。
- **居場所:** 河口付近の砂泥地、浅瀬から深場のブレイクライン。
- **主要ポイント:** 都田川河口、庄内湖、舘山寺、内浦湾。

## 3. メインベイト（捕食対象）と集まる条件
- **多毛類（イソメ・ゴカイ）:** 砂泥に潜む虫エサ。
- **集まる条件:** 汽水域で水深が浅く、かつプランクトンが豊富なエリア。

## 4. 最も有効なアプローチ
- **のべ竿:** ミャク釣りやウキ釣り。手返し良く数釣りが楽しめる。
- **チョイ投げ:** 10月以降、深場へ移動したデカハゼを狙う際に有効。
- **エサ:** 石ゴカイを小さく切って使用。ホタテの貝柱なども実績あり。
- **コツ:** 底をトントンと叩くように誘い、ハゼを寄せる。
