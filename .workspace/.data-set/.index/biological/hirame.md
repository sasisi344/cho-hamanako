# 浜名湖リサーチデータ：ヒラメ（晩秋〜冬）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「接岸・産卵前荒食い期」**。
- **活性:** 10月〜1月。特に11月以降、海水温の低下とともにベイトフィッシュ（アジ・イワシ・サッパ等）を追って積極的に浅場へ入ってくる。
- **特徴:** 「座布団ヒラメ」と呼ばれる70cmオーバーの期待も高い。

## 2. 適水温と魚の居場所の推移
- **適水温:** 15℃〜20℃。
- **居場所:** 澪筋の斜面、今切口周辺、外海と接するエリアの砂地。
- **主要ポイント:** 新居海釣り公園、網干場、砂揚げ場、舞阪サーフ。

## 3. メインベイト（捕食対象）と集まる条件
- **小魚（アジ・イワシ・サッパ）:** 最も好むベイト。
- **集まる条件:** 潮流が激しく、かつベイトが追い込まれる場所（ブレイクや障害物の際）。

## 4. 最も有効なアプローチ
- **泳がせ釣り:** サビキで釣ったアジやイワシを活き餌として使用。
- **ルアー:** ボトムワインド、ジグヘッド＋ワーム。ボトムを叩くように誘う。
- **コツ:** 浜名湖の速い潮を利用し、ベイトが溜まる「ヨレ」をピンポイントで通す。
