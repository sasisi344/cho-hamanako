# 浜名湖リサーチデータ：カレイ（冬〜春）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「産卵接岸期・乗っ込み」**。
- **活性:** 12月〜2月がピーク。産卵のために外洋から湖内の深場やミオ筋に接岸し、体力を蓄えるために捕食活動を行う。
- **特徴:** 「冬の女王」と呼ばれ、浜名湖の冬の投げ釣りを代表するターゲット。数釣りよりも一発大物の期待が高い。

## 2. 適水温と魚の居場所の推移
- **適水温:** 10℃〜15℃程度。低水温を好む。
- **居場所:** 浜名湖南部の**ミオ筋（船の通り道）**の縁や、水深のある砂地。
- **主要ポイント:** 砂揚げ場、新居海釣り公園、舞阪堤防、1番〜3番ミオ周辺。

## 3. メインベイト（捕食対象）と集まる条件
- **多毛類（イソメ・ゴカイ）:** 底に潜む虫エサ。
- **甲殻類:** 小さなエビやカニ。
- **集まる条件:** 潮通しが良く、かつ餌が溜まりやすい砂泥地のカケアガリ。

## 4. 最も有効なアプローチ
- **投げ釣り:** 25号〜30号の重めのオモリを使用。浜名湖の速い潮に流されないようスパイク天秤も有効。
- **エサ:** 青イソメを複数匹房掛けにする「アピール重視」や、赤蛇虫（ジャムシ）を使用。
- **時間帯:** 日中の潮止まり前後が最大のチャンス。夜釣りでも実績があるが、日中の方が狙いやすい。
