# 浜名湖リサーチデータ：カサゴ（通年）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「居着き・産卵期（冬）」**。
- **活性:** 通年狙えるが、冬の12月〜3月が数釣りの最盛期。
- **特徴:** 浜名湖で最も手軽に釣れる根魚。数・型ともに安定している。

## 2. 適水温と魚の居場所の推移
- **適水温:** 15℃〜25℃（幅広い）。
- **居場所:** テトラの隙間、敷石、ケーソンの穴、船の下。
- **主要ポイント:** 新居海釣り公園、舞阪港、網干場、砂揚げ場。

## 3. メインベイト（捕食対象）と集まる条件
- **甲殻類:** 小さなエビやカニ。
- **小魚:** ハゼ、キス、アジの稚魚。
- **集まる条件:** 複雑な障害物があり、適度な流れの供給がある場所。

## 4. 最も有効なアプローチ
- **穴釣り:** ブラクリ仕掛け。エサはサバの切り身や青イソメ。
- **ライトロック:** ジグヘッド＋クロー系ワーム。底を叩く。
- **コツ:** 1つの穴で反応がなければ即座に移動する「ラン＆ガン」が有効。
