# 浜名湖リサーチデータ：キビレ（四季）

## 1. 生態ステータスと活性
- **春（3月〜5月）: 「入湖期・入っ込み」**。外海から湖内へ入ってくる群れ。活性が高い。
- **夏（6月〜8月）: 「高活性期」**。高水温を好み、シャローに広く分散する。トップへの反応が良い。
- **秋（9月〜11月）: 「産卵期（乗っ込み）」**。メインの産卵シーズン。荒食いし、数・型ともに期待できる。
- **冬（12月〜2月）: 「越冬・バチ抜け期」**。産卵を終えて深場へ移動。2月からバチ抜けとともに活性が上がる。

## 2. 適水温と居場所の推移
- **適水温:** 12℃〜28℃（クロダイより低水温に強く、冬でも釣果が出やすい）。
- **春:** 今切口から中浜名湖、奥浜名湖へと徐々に広がる。
- **夏:** 水深1m以下のシャローフラット、特に砂泥地を好む。
- **秋:** 湖内各所の産卵場（深場に隣接するシャロー）に集まる。
- **冬:** 中浜名湖〜庄内湖南部の水深があるオープンエリア。中ノ島周辺など。

## 3. メインベイト（捕食対象）
- **春:** バチ、ハク、エビ。
- **夏:** ハク、イナッコ、カニ、エビ。
- **秋:** エビ、カニ、小魚。
- **冬:** バチ（2月以降）、落ちハゼ、甲殻類。

## 4. 最も有効なアプローチ
- **ボトムゲーム（チニング）:** 年間を通じて有効。特にフリーリグでのズル引き＆ステイが基本。
- **トップゲーム:** 6月〜9月。クロダイよりもアグレッシブにルアーを追う傾向がある。
- **電気ウキ釣り:** 夜釣りの大物狙い。エサは青イソメやボケ。
- **バチ抜けパターン:** 2月〜4月の夜間。シンキングペンシルのデッドスロー巻き。
