# 浜名湖リサーチデータ：シロギス（春〜秋）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「接岸・産卵期」**。
- **活性:** 5月〜11月。水温上昇とともに浅場へ移動。夏から秋にかけてが数釣りの最盛期。
- **特徴:** 砂地の女王。浜名湖全域の砂地エリアで狙える。

## 2. 適水温と魚の居場所の推移
- **適水温:** 20℃〜25℃。
- **居場所:** 浜名湖南部の砂地エリア、堤防周りの浅場。
- **主要ポイント:** 湖南部全域、新居海釣り公園、砂揚げ場、乙女園周辺。

## 3. メインベイト（捕食対象）と集まる条件
- **多毛類（イソメ・ゴカイ）:** 砂の中に潜む虫エサ。
- **集まる条件:** 砂地の底質で、波打ち際やブレイクライン。

## 4. 最も有効なアプローチ
- **チョイ投げ:** 5号〜10号のテンビン仕掛け。広く探る「引き釣り」が基本。
- **エサ:** 石ゴカイ。鮮度が重要で、細かく動くものが良い。
- **コツ:** 砂紋（砂底の凹凸）を感じながら、ゆっくりと仕掛けを引いてくる。
