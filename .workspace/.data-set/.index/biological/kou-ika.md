# 浜名湖リサーチデータ：コウイカ（春〜初夏）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「産卵接岸期」**。
- **活性:** 3月〜6月。産卵のために湖内の砂地へ移動し、底付近を徘徊する。
- **特徴:** 浜名湖ではエギングで手軽に狙えるターゲット。肉厚で美味。

## 2. 適水温と魚の居場所の推移
- **適水温:** 15℃〜20℃。
- **居場所:** 水深のある砂地の底、澪筋付近。
- **主要ポイント:** 砂揚げ場、新居海釣り公園、網干場、乙女園周辺。

## 3. メインベイト（捕食対象）と集まる条件
- **甲殻類:** 小エビやカニ（底にいるもの）。
- **集まる条件:** 砂地の底質で、産卵に適した障害物（カキ殻や石）が点在するエリア。

## 4. 最も有効なアプローチ
- **エギング:** 底をタイトに狙う。ボトムをズル引きする感覚。
- **コツ:** 派手なアクション（シャクリ）よりも、底を小突きながらの「ステイ（待機）」を長めに取ること。
