# 浜名湖リサーチデータ：クロダイ（秋・冬）

## 1. 生態ステータスと活性
- **秋（9月〜11月）: 「落ち・荒食い期」**。冬の深場移動を控え、体力を蓄えるために活発に捕食する。数・型ともに年間で最も狙いやすい。
- **冬（12月〜2月）: 「越冬期」**。低水温により代謝が下がり、深場（ミオ筋等）に身を寄せる。活性は低いが、特定のポイント（砂揚場など）ではフカセ釣りが成立する。

## 2. 適水温と魚の居場所の推移
- **秋:** 水温低下とともに魚が湖奥から湾口部（表浜名湖）へ移動する。
    - **9月〜10月:** まだシャローに残る個体も多く、河川内や水門周りでも大型が狙える。
    - **11月:** 本格的に南下し、新居海釣り公園や舞阪堤防などの潮通しの良いエリアに集結する。
- **冬:** 
    - 水深のある**ミオ筋のブレイク**や、水温が安定しやすい**砂揚場（新居）**の深場に定着する。
    - 北西の強風（遠州の空っ風）を避けられる風裏の深場が重要。

## 3. メインベイト（捕食対象）
- **秋:** カニ、エビ、ボケ、小魚（イナッコ、コノシロなど）。
- **冬:** 底生生物（エビ、カニ）、落ちハゼ、アミ類。

## 4. 最も有効なアプローチ
- **秋:** 
    - **ウキダンゴ釣り（紀州釣り）:** エサ取りをかわしつつ、効率よく底を狙える。
    - **チヌトップ:** 10月初旬までは日中のシャローで高反応。
    - **ブッコミ釣り:** 夜釣りの大物狙いに最適。エサはカニやボケ。
- **冬:** 
    - **フカセ釣り:** 棒ウキを使用した繊細な仕掛け。砂揚場が聖地とされる。
    - **ボトムチニング:** 深場に隣接するシャローの駆け上がりをワームでスローに攻める。
