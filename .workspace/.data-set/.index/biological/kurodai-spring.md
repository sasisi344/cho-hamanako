# 浜名湖リサーチデータ：クロダイ（春・3月〜5月）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「乗っ込み（産卵前）」**の最盛期。
- **活性:** 非常に高い。産卵に向けて体力を蓄えるために大型の個体が浅場へ接岸し、積極的に餌を追う「荒食い」が見られる。一年で最も「年無し（50cmオーバー）」を仕留める確率が高い時期。
- **特徴:** 3月は個体数が徐々に増え始め、4月〜5月にかけて広範囲の浅場で安定した釣果が出るようになる。

## 2. 適水温と魚の居場所の推移
- **適水温:** **13℃〜18℃**。13℃が接岸のスイッチとなる。
- **3月（初期）:** 水温が安定しやすい**表浜名湖（舞阪・新居）**の深場に隣接したブレイクや、水温上昇が早い**奥浜名湖（都田川河口付近）**のシャローに二極化する。
- **4月〜5月（盛期）:** 水温が15℃を超えて安定すると、湖内全域の**アマモ帯（藻場）**や、カキ棚・石畳などの障害物周りに広く分散する。特に潮通しの良いミオ筋（船の通り道）沿いのシャローフラットが有望。

## 3. メインベイト（捕食対象）と集まる条件
- **ハク（ボラの稚魚）:** 3cm〜5cm程度の群れが岸際に溜まるエリア。特に水温が高い日のシャロー。
- **バチ（多毛類）:** 4月前後の大潮・中潮の夜間、下げ潮に乗って流れるタイミング。
- **稚鮎:** 春に海から遡上するタイミングで今切口から湖内へ入るルート。
- **底生生物（エビ・カニ・ボケ）:** 定番ベイト。カキ殻や石の間、砂泥地に潜む甲殻類。

## 4. 最も有効なアプローチ
### ルアーフィッシング（チニング）
- **稚鮎・ハクパターン:** 50mm〜70mmの小型シンキングペンシルやシャローミノー。デッドスロー巻きや微細なトゥイッチが有効。
- **ボトムパターン:** フリーリグ（5g〜10g）に2インチ〜3インチのクロー系ワーム。石畳やカキ棚の際をズル引き＆ステイ。

### 餌釣り
- **フカセ釣り:** 配合エサでポイントを切り開き、オキアミやコーン、サナギを使用。表浜名湖の堤防周りがメイン。
- **ブッコミ釣り:** 浜名湖最強エサの一つ**「ボケ」**を使用。砂泥地のポイントで大型の実績が高い。

### 時間帯・条件
- **時間帯:** 朝夕のマヅメ時がベストだが、春は日中の水温上昇のタイミング（昼過ぎ〜夕方）も活性が上がりやすい。
- **潮回り:** 上げ潮に乗って魚が浅場に差し込み、下げ潮でベイトが流されるタイミングがチャンス。
