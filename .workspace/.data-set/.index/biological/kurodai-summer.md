# 浜名湖リサーチデータ：クロダイ（夏・6月〜8月）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「高活性期・トップゲーム期」**。産卵を終えた個体（アフタースポーン）が体力を回復し、高水温とともに代謝が上がって水面近くの餌を積極的に追う時期。
- **活性:** 非常に高い。特に日中の日差しが強く、水温が上がりきるタイミングで「トップ（水面）」への反応が爆発する。
- **特徴:** 浜名湖は「チヌトップ発祥の地」の一つとされ、全国的にも魚影が濃く、視覚的に楽しめるエキサイティングなシーズン。

## 2. 適水温と魚の居場所の推移
- **適水温:** **22℃〜30℃**。25℃を超えるとトップへの反応が安定する。
- **居場所:** 水深30cm〜1.5m程度の**シャローフラット（浅瀬の平地）**がメイン。
- **ストラクチャー:** 砂地のフラットの中に点在する**カキ根（牡蠣殻の山）**、杭、ブレイク（かけ上がり）の肩、橋脚のシェード（陰）。
- **エリア:** 
    - **奥浜名湖（庄内湖など）:** 水温が上がりやすく、6月上旬からいち早く反応が出始める。
    - **表浜名湖:** 潮通しが良く、真夏の猛暑日でもフレッシュな水が入るエリアに回遊個体が集まる。

## 3. メインベイト（捕食対象）と集まる条件
- **ハク・イナッコ（ボラの幼魚）:** 岸際のシャローに群れて逃げ惑う小魚。トップゲームの最重要ベイト。
- **稚鮎:** 初夏にかけて湖内を回遊する個体。
- **エビ・カニ:** シャローの底に潜むが、これらを探して意識が上を向いている個体が多い。
- **昆虫・フナムシ:** 護岸沿いやオーバーハング下で水面に落ちてくるものを捕食。

## 4. 最も有効なアプローチ
### トップウォーターゲーム（ルアー）
- **ペンシルベイト:** ドッグウォークアクションで逃げるベイトを演出。浜名湖では見切られないよう、やや速めのテンポが有効。
- **ポッパー:** ポップ音と飛沫で魚を寄せる。風がある時や、広範囲から魚を呼びたい時に強力。
- **ルアーカラー:** 澄み潮ではクリア系が基本。雨後の濁りや光量が少ない時は、アカキンやチャートなどの派手な色が有効。

### ボトムゲーム（チニング）
- **フリーリグ:** トップに反応が悪い時や、日中の中層に浮いていない個体を狙う際に使用。2.5インチ〜3インチのクローワーム。

### 時間帯・条件
- **時間帯:** 朝夕のマヅメ時はもちろん、真夏の**「ドピーカン（快晴・日中）」**でも爆発的な釣果が出ることが多い。
- **潮回り:** 潮が動いて水位が変動するタイミング。特に「上げ潮」でシャローに魚が差し込んでくるタイミングがチャンス。
- **注意点:** ウェーディングの際は**アカエイ**の危険があるため、エイガードの装着や「すり足」での移動が必須。
