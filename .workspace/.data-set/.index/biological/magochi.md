# 浜名湖リサーチデータ：マゴチ（春〜初夏・晩秋）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「接岸・照りゴチ期」**。
- **活性:** 5月〜7月（照りゴチ）。産卵のために浅場へ移動し、高水温とともに活性が爆発する。
- **特徴:** ヒラメと並ぶフラットフィッシュの人気ターゲット。10月〜12月にも第2のピーク（落ち期）がある。

## 2. 適水温と魚の居場所の推移
- **適水温:** 20℃〜25℃。
- **居場所:** 1番〜3番の澪筋（ミオ筋）、砂地のシャローエリア、アマモ帯周辺。
- **主要ポイント:** 甕ヶ渕、中之島周辺、今切口周辺。

## 3. メインベイト（捕食対象）と集まる条件
- **小魚（ハゼ・キス・アジ）:** 浅場に群れる小魚を好む。
- **甲殻類:** エビやカニ。
- **集まる条件:** 砂地の底質で、適度な潮流と小魚の供給があるブレイクライン。

## 4. 最も有効なアプローチ
- **ルアー:** シャッドテールワームやバイブレーション。ボトムから少し浮かせて誘う。
- **投げ釣り:** 活きたハゼやキスをエサにする泳がせ釣り。
- **コツ:** 非常に硬い口を持つため、力強いアワセ（フッキング）が必須。
