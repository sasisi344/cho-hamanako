# 浜名湖リサーチデータ：メバル（冬〜春）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「接岸・産卵期」**。
- **活性:** 12月〜4月。水温が10℃〜15℃程度の時期に活性が上がり、夜間に水面近くへ浮いてくる。
- **特徴:** 「春告魚」と呼ばれ、冬のライトゲームの華。

## 2. 適水温と魚の居場所の推移
- **適水温:** 10℃〜15℃。
- **居場所:** 澪筋の肩、敷石の際、アマモ帯、橋脚の明暗部。
- **主要ポイント:** 弁天島海浜公園、新居海釣り公園、中之島周辺。

## 3. メインベイト（捕食対象）と集まる条件
- **多毛類（バチ）:** 2月以降のバチ抜けとともに。
- **アミ類:** プランクトンの集合体。
- **小魚（ハク）:** 3月以降。
- **集まる条件:** 潮流が適度にあり、身を隠せるストラクチャーと光の明暗がある場所。

## 4. 最も有効なアプローチ
- **メバリング:** ジグヘッド（0.5g〜1.5g）にワーム、またはプラグ。
- **のべ竿:** 堤防の際でシラサエビをエサにするウキ釣り。
- **コツ:** スローに「ただ巻き」することが基本。夜の常夜灯の境目が狙い目。
