# 浜名湖リサーチデータ：メジナ（グレ）（通年）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「成長・回遊期」**。
- **活性:** 5月〜11月（高活性期）。通年狙えるが、水温が高い時期は数釣りが楽しめる。
- **特徴:** 浜名湖のメジナは20cm〜30cmサイズが主流。非常にパワフルな引きが魅力。

## 2. 適水温と魚の居場所の推移
- **適水温:** 18℃〜25℃。
- **居場所:** 潮通しの良い堤防の際、テトラ帯、澪筋のブレイク。
- **主要ポイント:** 新居海釣り公園、今切口周辺、舞阪堤防、網干場。

## 3. メインベイト（捕食対象）と集まる条件
- **海藻:** 堤防の際や石に付く藻。
- **甲殻類:** 小さなエビやカニ。
- **アミエビ・オキアミ:** フカセ釣りで撒かれるエサ。
- **集まる条件:** 潮通しが良く、酸素供給が豊富なエリア。

## 4. 最も有効なアプローチ
- **ウキフカセ釣り:** 配合エサで魚を寄せ、オキアミをサシエにする。
- **ルアー:** 稀にマイクロプラグで狙うことも可能だが、エサ釣りが圧倒的に有利。
- **コツ:** 浜名湖の速い潮の中で仕掛けを同調させることが重要。
