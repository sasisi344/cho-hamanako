# 浜名湖リサーチデータ：シーバス（スズキ・セイゴ・マダカ）

## 1. シーズナルパターン
- **春（2月〜5月）: 「バチ抜け・ハクパターン」**。産卵を終えた個体が体力回復のため入湖。ベイトが小さく「悶絶パターン」になることも多い。
- **夏（6月〜8月）: 「イナッコパターン」**。水温上昇とともに活性化。河川（都田川など）へ遡上する個体も多い。
- **秋（9月〜11月）: 「ハイシーズン・コノシロ・落ちアユ」**。冬の産卵に向けた荒食い期。大型（ランカー）が最も狙いやすい。
- **冬（12月〜1月）: 「落ち・越冬期」**。水温低下で産卵のため外洋へ。表浜名湖の深場にターゲットが絞られる。

## 2. ベイトパターン
- **バチ:** 2月中旬〜5月の大潮前後の夜間。シンキングペンシルが特効。
- **ハク（ボラ稚魚）:** 3cm以下の群れ。攻略が難しい。
- **イナッコ（ボラ幼魚）:** 5cm〜10cm。夜間のボイル狙いが主体。
- **コノシロ・サッパ:** 秋の大型ベイト。12cm以上のミノーやビッグベイトも有効。
- **落ちアユ:** 晩秋、増水時や産卵後の流下アユ。

## 3. 適水温とポイント
- **適水温:** 12℃〜25℃（20℃前後が最も活性が高い）。
- **春:** バチ抜けに伴い表から奥へと北上。水深の浅いポイントの夜間が狙い目。
- **夏:** 流れのあるミオ筋、河川の橋脚、水深のある中之島周辺。
- **秋:** ベイト（コノシロ等）の接岸エリア。南下する群れを今切口周辺で待ち伏せる。
- **冬:** 水温が安定しやすい表浜名湖の深場（ミオ筋沿い）。

## 4. 最も有効なアプローチ
- **シンキングペンシル:** 春のバチ抜け、夏のイナッコパターンのデッドスロー。
- **シャローミノー:** 秋の荒食い期。地形変化や流れの筋を通す。
- **バイブレーション:** 夏の日中、ミオ筋や橋脚をリアクションで攻める。
- **ドリフト釣法:** 河川や潮の効いたエリアで、ルアーを流れに乗せて漂わせる。
