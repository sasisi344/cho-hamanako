# 浜名湖リサーチデータ：マダコ（初夏〜夏）

## 1. 現在の生態ステータスと活性
- **ステータス:** **「接岸・成長期」**。
- **活性:** 5月下旬〜8月。産卵を控えた個体や成長途中の個体が浅場へ集結。
- **特徴:** 全国でも有数のタコ釣りポイント。6月の数釣りと8月の型狙いが楽しめる。

## 2. 適水温と魚の居場所の推移
- **適水温:** 20℃〜25℃。
- **居場所:** 砂地と岩礁・敷石が混在するエリア。壁際や基礎。
- **主要ポイント:** 砂揚げ場、新居海釣り公園、網干場、六角堂周辺。

## 3. メインベイト（捕食対象）と集まる条件
- **甲殻類（カニ・エビ）:** タコの主食。
- **貝類:** アサリやカキ。
- **集まる条件:** 砂地でかつ餌（カキ棚や石組み）が豊富なエリア。

## 4. 最も有効なアプローチ
- **タコエギ:** エギングの要領で底を叩いて誘う。
- **タコテンヤ:** カニやアジをエサにする。
- **コツ:** 浜名湖の速い潮で底が取れない時は、30号以上のオモリを足してでも底を確保すること。
