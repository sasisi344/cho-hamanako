# 浜名湖リサーチデータ：季節×スタイル別 最適ポイント・ロジック

## 1. 春（2月〜5月）

### バチ抜け攻略 (シーバス・キビレ)
- **時期:** 2月中旬〜4月。大潮・後中潮の夜間、満潮からの下げ始め。
- **ポイント:**
    - **初期（2月）:** 表浜名湖（南エリア）。渚園周辺、サクラマル、今切口周辺。水温が安定しやすい場所から始まる。
    - **最盛期（3-4月）:** 湖全体へ拡大。庄内湖（はまゆう大橋周辺）、都田川河口、瀬戸水道。
- **理由:** カキ殻帯や砂泥地に生息するゴカイ類が産卵のために浮上する「バチ抜け」に魚が狂う。
- **コツ:** 細身のシンキングペンシルを水面直下でデッドスロー。

### 乗っ込み攻略 (クロダイ・キビレ)
- **時期:** 3月〜5月。
- **ポイント:** 
    - **表浜名湖:** 新居海釣り公園、舞阪堤防（外海から入る個体）。
    - **奥浜名湖:** 都田川河口、猪鼻湖周辺（水温が上がりやすく、産卵を控えた大型が集まる）。
- **理由:** 産卵のために浅場へ接岸する「乗っ込み」個体を狙う。

## 2. 夏（6月〜8月）

### チヌトップゲーム (クロダイ・キビレ)
- **時期:** 6月〜8月（最盛期は7-8月）。日中の快晴時。
- **ポイント:** 中之島周辺、内浦湾、村櫛海岸などの広大なシャローフラット。水深30cm〜1m。
- **理由:** 高水温で代謝が上がり、水面を意識する個体が増える。浜名湖特有の広大な浅瀬が主戦場。

### 初夏のタコング (マダコ)
- **時期:** 5月下旬〜8月。
- **ポイント:** 新居海釣り公園、砂揚げ場（浜名港）、網干場。
- **理由:** この時期に数釣りが楽しめるターゲット。壁際や基礎の石周りに潜む。
- **コツ:** 激流に対応するため30号以上のオモリを併用。

### ファミリー向けサビキ釣り (アジ・サバ・イワシ)
- **時期:** 6月〜9月。
- **ポイント:** 
    - **新居海釣り公園:** 駐車場・トイレ完備で随一の人気。T字堤の内側は波が穏やか。
    - **砂揚げ場:** 車横付け可能。
- **理由:** 群れの回遊が安定し、子供でも鈴なりで釣れる。

## 3. 秋（9月〜11月）

### 落ち攻略 (クロダイ・シーバス・キビレ)
- **時期:** 9月〜11月。
- **ポイント:** 新居海釣り公園、砂揚げ場、舞阪堤防、今切口。
- **理由:** 冬の深場移動（落ち）を控えた荒食いシーズン。魚が湖内から湾口部へ集結する。

### ハゼ釣り
- **時期:** 8月〜11月。
- **ポイント:** 都田川河口（聖地）、庄内湖（花川周辺）、舘山寺内浦。
- **理由:** 8-9月は浅場で数釣り、10-11月はサイズアップした「落ちハゼ」を狙える。

## 4. 冬（12月〜2月）

### 寒カレイ攻略
- **時期:** 12月〜2月。
- **ポイント:** 海南部のミオ筋（船の通り道）周辺、新居海釣り公園、舞阪堤防。
- **理由:** 産卵のために接岸する「冬の女王」を置き竿でじっくり狙う。

### 厳寒期のロックフィッシュ (カサゴ・メバル)
- **時期:** 12月〜3月。
- **ポイント:** 新居海釣り公園のテトラ・敷石、舞阪漁港周辺、弁天島海浜公園（常夜灯周り）。
- **理由:** 他の魚種が釣れない厳寒期でも安定して釣れる。夜のライトゲームが主流。
