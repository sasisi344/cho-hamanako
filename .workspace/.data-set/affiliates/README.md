# アフィリエイト商品リサーチデータ（data-set/affiliates）

> **このディレクトリの目的**
> 魚種・釣法ごとのアフィリエイト商品候補データ（Amazonリンク＋選定理由メモ）を格納する。
> 実際にサイトのTackleCardで使用する正式データは `src/content/affiliates/` の `.yaml` ファイルで管理される。
> ここにあるのは「素材・リサーチ段階のメモ」である。

## 使い方ガイド

### 新しいTackleCard用YAMLを作る場合
1. このディレクトリの該当魚種 `.md` を参照して商品候補を確認する
2. Amazonリンクと選定理由・スペックを確認する
3. `src/content/affiliates/{category}/{product-slug}.yaml` に正式なYAMLデータを作成する

### 共通アイテムの判定基準
- **2つ以上の魚種カテゴリで使われるアイテム** → `common/` カテゴリに格納
- 例: 3000番スピニングリール、万能磯竿、ヘッドライト、ウェーダー等

### アフィリエイトタグ
- **Amazonアソシエイト**: `tag=sasisi344-22`
- **A8.net**: 車中泊マット、ポータブル電源等で使用

## 注意事項
- `.md` ファイルはAstroのコンテンツコレクションには含めないこと（MixedContentDataCollectionErrorの原因になる）
- 正式なアフィリエイトデータは `src/content/affiliates/` の `.yaml` ファイルのみ
- このディレクトリのデータは「リサーチ素材」であり、最終データではない
