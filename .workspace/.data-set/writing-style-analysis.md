# @344dev ライティングスタイル分析レポート

本レポートは、`.workspace/.archieve` に保管されている過去記事群から、管理人（@344dev）独自のライティングスタイル、構造的特徴、および価値観を抽出し、今後のAIによる記事生成（Astro Sphereへの移行・リライト）における指針を定義するものです。

---

## 1. 🎭 ペルソナとトーン（声の調子）

### 「頼れる地元のベテラン釣り仲間」
単なる情報の羅列ではなく、隣でアドバイスしてくれているような「親近感」と、長年の経験に裏打ちされた「信頼感」が共存しています。

- **文体（です・ます調）**: 丁寧ながらも堅苦しくなく、ブログ的な親しみやすさを重視。
- **ポジティブな呼びかけ**: 冒頭や結びに「ぜひ〜してください！」「楽しみましょう！」といった、読者のモチベーションを高めるフレーズを多用。
- **主観と客観のブレンド**: 「私の経験上では」「メソッド的に言うと」など、一般論と実体験を明確に使い分け、説得力を高めています。

## 2. 📝 言語的・文体的特徴

### リズムと可読性
- **短〜中程度の文章**: 1文が長くならず、スマホでの閲覧を意識した適度な改行が見られます。
- **接続詞の活用**: 「さて」「ですが」「要するに」「必然的に」など、論理構成を分かりやすく導く言葉が効果的に配置されています。

### 語彙・専門用語
- **浜名湖特化の地名/用語**: 「サクラマル」「網干場」「ホトニクス下」など、地元民しか知らないような特定スポットを具体名で出すことで圧倒的なリアリティを演出。
- **魚種ごとの呼称**: クロダイの幼魚を「チンタ」と呼ぶなど、地域の釣り用語を自然に使用。
- **カタカナ用語のバランス**: 「ウェーディング」「ボトムワインド」などの専門用語は使いつつ、その直後に平易な解説を加える配慮があります。

## 3. 🏗️ 記事の構造的DNA（テンプレート化された構成）

大半の記事が以下の「黄金構成」に従っています。

1. **アイキャッチ・導入**: 季節感やターゲットの魅力を伝え、読者の悩みに寄り添う。
2. **ポイント紹介（エリア別）**: 「表・中・奥」などのエリア分けと、具体的な実績ポイントのリストアップ。
3. **タックル・仕掛け解説**: 初心者向け・本格派向けの2段構えで、具体的な製品例（ダイワ、シマノ等）を提示。
4. **季節・時間の攻略法**: 潮汐や水温、月別の動きなど、理論に基づいた攻略法。
5. **食・レシピ（重要）**: 釣った後の楽しみ（塩焼き、煮付け、洋風アレンジなど）を必ず記述。「食害」などの環境側面にも触れる。
6. **まとめ**: 箇条書きで要点を再確認し、安全とマナーの呼びかけで締める。

## 4. 💎 核心的な価値観（ポリシー）

ライティングの根底には以下の3つの柱があります。

- **「再現性」の追求**: 初心者が行っても本当に釣れるよう、棚の深さ（1.5m等）や潮汐のタイミングを具体的に数値化。
- **「コストパフォーマンス」への配慮**: 高級タックルだけでなく、ダイソーや入門セットでの楽しみ方にも等しく敬意を払う。
- **「安全とマナー」の優先**: 毒魚への注意、ライフジャケットの推奨、地元のルール（浜名漁協など）の遵守を欠かさない。

## 🤖 AIエージェントへの指示（Prompt）への反映案

今後、記事を作成・リライトする際は以下のガードレールを設定してください。

```markdown
- **専門性**: 浜名湖独自の地形（澪筋、浅瀬の広がり）を考慮した記述を行う。
- **具体性**: 「場所による」で濁さず、具体的なポイント名（archive内の名称）を引用する。
- **親しみやすさ**: 読者を「初心者」と想定し、専門用語には補足を入れつつ、挑戦を促す温かい言葉をかける。
- **構成**: H2で大項目、H3で詳細、最後は必ず「箇条書きのまとめ」と「マナーの呼びかけ」を入れる。
- **食へのこだわり**: 魚種に合わせた具体的な「和・洋」のレシピ案を1セクション設ける。
```
