1.  **ターゲット選定**: 各ポイントの柱となる魚種を3つ選ぶ。
2.  **データ抽出**: [リサーチ・ガイド](file:///c:/Users/sasis/344dev/cho-hamanako/.workspace/_point-pillar-migration/catch-chart-research-guide.md) のプロンプトを使用。
3.  **データ入力**: 1月〜12月の報告数（爆釣・好調・散発・圏外）を確定させる。
4.  **反映**: クラスター記事（深掘りガイド）の「釣果月別グラフ」セクションへ。
