# クエリから既存記事の強化を狙う

## GSC12ヶ月データを取得
- 格納先
[.workspace\access-data\gsc-data]

## まずやること
- 1年間のクエリデータから、表示回数は多いがクリック数が少ない「改善見込みクエリ」を発掘。既存記事をリライトして対処するか、新規記事を作成して取りこぼしを防ぐかの施策をする。

### 手動で上位を探った結果
- 新居海釣公園 ライブカメラ（浜名湖 海釣り公園 ライブカメラ）
表示回数5500回に対しクリックが29回ともったいない。上位を分析して上回るコンテンツにする。既存コンテンツに類似しているのがあるかどうか。

個人的な感想として、台風シーズンになると閲覧があがると考えている。

- 浜名湖釣り　車　横付け
表示1230に対してクリック191と悪くないが、まだ改善する余地はあるはず。

- 庄内湖
表示900に対してクリック5と割合が少ない。表示が上位だけあってもったいない。

- 浜名湖 アジング
表示637に対してクリック48。悪すぎるわけじゃないが、アジング部門は強みがあるので、分散しているコンテンツを集約するのが解決方法になるかも。

- 近くの釣具店
表示630でクリック2。クエリKW的にローカル情報が求められるので、必ずしも浜名湖の釣具店と限らないから判断が難しい。

- 表浜名湖
表示476にクリック3。KWから観光目的も強いだろうから、浜名湖の歴史をまとめたページを作成、もしくは既存で類似ページがあるならコンテンツを充実させる。釣りと観光で強みを出す。

- 浜名湖 ハゼ釣り 新川
表示436でクリック13。浜名湖のハゼ釣りは都田川が有名で、そこのクリックはとれている。新川はマイナーポイントだから認知度が必ずしも広くない。ポイント記事であるはずなので、より深堀りした記事内容にする。

合わせて「浜名湖ハゼ釣りポイント」のクエリにも対応する。

- 浜名湖 水深
表示401でクリック9。浜名湖全体の水深図についての記事をチェック。なぜユーザーは水深を調べたいのか、そこのベネフィットを考える。

- 浜名湖 今切口 釣り禁止
釣り禁止エリアについては既存記事がある。より詳細な情報を提供することで、他との差別性を図るのが妥当か。

## 総合的な考察
「ライブカメラ」のクエリKWが検索結果で分散している。「海釣公園、海釣り公園」と表記ブレがあるので、どちらにも対応するならフロントマターにmeta keywordで対応するのがベストか。記事中では表記ブレは許されないので、公式情報の「新居弁天海釣公園」で統一するべき。
