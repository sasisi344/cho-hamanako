# Tactics カテゴリ拡充プラン

**作成日**: 2026-07-14
**現状**: `tactics/method-kayaking-intro` 1本のみ（カヤック・SUPフィッシング入門）
**方針**: エリア攻略（浜名湖を区域別に分け、既存ポイント記事とリンクする）＋釣り方系の2軸で拡充

---

## 背景と設計思想

個別ポイント記事（`points/omote/`, `points/naka/`, `points/oku/`）は豊富だが、
「浜名湖の猪鼻湖エリアをどう攻略するか」「中浜名湖でボートを使うとどう変わるか」
といった**エリアを俯瞰した戦略記事**が不在。

- ポイント記事は「一点」を深掘り → tactics 記事は「一帯」を繋ぐ中間レイヤー
- 読者の回遊パス: エリア攻略記事 → 個別ポイント BlogCard → target/fukabori記事
- GSC 観点: 「猪鼻湖 釣り」は既存上位KW。エリア攻略記事を足すことでブランドワード化を狙う

**構成案**:
1. 表浜名湖エリアの全体像（今切口〜弁天島〜砂揚げ場〜渚園の地形・潮流・ミオ筋）
2. ショアポイントマップ（BlogCard で各ポイントへ）
3. **プレジャーボートフィッシング概論**（水深・ミオ筋・陸っぱりとの比較優位）
   - 砂揚げ場〜渚園間の航路（ミオ筋）でタコ・マゴチが人気
   - ボートレース浜名湖周辺はショアプレッシャー低でサヨリ・マゴチ実績
   - 弁天島南シャローはゴムボートでのチヌトップも可能
4. ⚠️ ボート利用上の法令・航行ルール（必須セクション）
5. 季節別ターゲット（春クロダイ → 夏タコ・青物 → 秋シーバス → 冬カレイ）
6. アクセス・駐車・近隣釣具店まとめ
7. まとめ

**⚠️ プレジャーボートに関する注意事項**（記事内に必ず盛り込む）:
- **今切口付近・内部（大橋以南の水路内）**: 航行は可能だが、**停泊・アンカリング・釣りは港則法等により禁止**。違反時は6ヶ月以下の懲役または10万円以下の罰金（`fishing-laws-hamanako.md` 参照）
- 今切口は大潮満潮前後に流速が川を超える激流になる。エンジン停止は即危険。初心者のエントリー禁止
- 砂揚げ場（新居漁港）は現役の漁港・商港。ボートでの無断係留・接近は厳禁

**ボートで狙える注目エリア**:
- **砂揚げ場〜渚園間の航路（ミオ筋）**: 足元タコ釣り（テンヤ・エギ）がボートから特に有効。砂泥底＋ミオ筋の流れがタコを集める。マゴチ・シロギスも通年ターゲット
- **ボートレース浜名湖周辺**: 陸っぱりからのアクセスが制限されるためプレッシャーが低い。サヨリ・カワハギ・マゴチの実績
- **弁天島〜渚園南のシャロー**: ゴムボート・SUP レベルで入れるチヌトップ・キビレエリア

## 実装上の注意

1. **カテゴリ**: 既存 `method-kayaking-intro` が `category: method` を使っているため、新記事も `method` に統一（URL は `tactics/` ディレクトリ内で良い）
2. **スラッグ確認**: BlogCard に使う全スラッグを `src/content/blog/` で実在確認してから記載
3. **事実確認が必要な情報**:
   - ボートレンタル店の営業状況・料金（変動注意）
   - SUP・カヤックの禁止区域・ライフジャケット義務（国交省・地元規制）
   - 漁業権区域（`fishing-laws-hamanako.md` 参照）
4. **method-kayaking-intro との差別化**: 入門記事と重複しないよう、「猪鼻湖特化」「中浜名湖ボート特化」でエリアを絞る
