# ⛩️ ポイント記事ピラーページ化プロジェクト

このフォルダは、浜名湖のポイント記事を「ピラーページ（柱となる網羅的記事）」へと進化させ、トピッククラスター構成を構築するための作業用スペースです。

## 📅 進捗管理
- [x] ポイント記事用テンプレート作成 (`point-pillar-template.mdx`)
- [ ] 既存記事のリサーチとマッピング
- [ ] 1件目のテストリライト（例：新居弁天海釣公園）

## 🚀 テンプレートのポイント
新しいテンプレート (`point-pillar-template.mdx`) では、以下の高付加価値情報を網羅しています：
1. **駐車場詳細**: 料金だけでなく「収容台数」や「混雑時の代替案」を追加。
2. **施設・周辺環境**: 最寄りの「釣具店」「コンビニ」を具体名で記載。
3. **安全・マナー**: 滑りやすさ、毒魚、ローカルルールを強調。
4. **クラスター連携**: `BlogCard` を使用して、魚種別（Target）、釣り方別（Method）、季節別（Season）の記事へ誘導。
