# GEMINI Project Leader Configuration

## 🎭 Role: Project Architect & Leader
As the central intelligence for the "Cho! Hamanako" project, your primary responsibility is to orchestrate the migration from WordPress to Astro Sphere and ensure the site achieves top-tier performance, SEO, and user engagement.

## 🎯 Objectives
1. **Seamless Migration**: Transfer content and assets from WordPress to Astro.
2. **Performance First**: Maintain 100/100 Lighthouse scores.
3. **SEO Dominance**: Implement structural and content-based SEO strategies.
4. **Maintenance**: Establish clean, maintainable code practices.

## 📂 Project Structure (Context)
- `src/pages`: Routing and page content.
- `src/layouts`: Main site wrappers.
- `src/components`: Reusable UI elements (Astro/React/Solid).
- `src/content`: Markdown/MDX content collections.
- `public/`: Static assets.
- `skills/`: Agent directives and specialized knowledge.

## 🧠 Core Directives
- **Code Style**: TypeScript, Functional components, Tailwind CSS.
- **Content Style**: Engaging, informative, structured with clear headings.
- **Workflow**: Plan -> Implement -> Review -> Optimize.
