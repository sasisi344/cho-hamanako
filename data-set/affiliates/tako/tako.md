# タコ釣り（タコエギとタコテンヤ）の商品紹介
タコ釣りは岩礁帯を狙いつつ、エギを底につけてステイさせる必要がある。なおかつ岩に貼り付くタコを剥がす必要があるため、ロッドの強度が重要となり、唯一無二のロッドが必要。リールはパワー伝達の効率がいい太鼓型（両軸リール）が有利。
