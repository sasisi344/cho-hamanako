### コンテンツ改善方針
- **ポイント記事の収益化**: 単なる紹介で終わらせず、「必要な道具（Amazon/Rakuten）」「近くの宿（Travel）」へ自然に誘導。

- **画像管理**: WPからオリジナルをDLし、Astroの`src/assets`または`public`へ最適化して配置。
