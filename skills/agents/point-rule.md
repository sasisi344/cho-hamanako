# 浜名湖釣りポイント表記統一ルール (point-rule)

## 📌 目的
浜名湖の釣りポイント名における表記揺れを防ぎ、サイト内検索の精度向上とユーザーの混乱回避を目的とする。

## 📍 優先表記リスト

| 読み | 優先表記 | 避けるべき不一致表記 | 補足 |
| :--- | :--- | :--- | :--- |
| すなあげば | **砂揚げ場** | 砂揚場 | 「げ」を送る表記で統一 |
| あみほしば | **網干場** | 網干 | 「場」まで含める |
| いまぎれぐち | **今切口** | 今切 | |
| あらいべんてん | **新居弁天海釣公園** | 海釣り公園、新居海釣公園 | 正式名称または「海釣公園」で識別 |
| まいさかてい | **舞阪堤** | 舞阪堤防 | 「堤」で統一 |
| いのはなこ | **猪鼻湖** | 猪花湖 | 漢字注意 |

## 📝 執筆時のガイドライン
1.  **初出時の配慮**: 記事内で最初に出現する際は、**砂揚げ場（すなあげば）** のように読みを併記することを推奨する。
2.  **内部リンク (BlogCard)**: スラッグ名と表示テキストに乖離がないよう、上記優先表記をベースにタイトルを構成する。
3.  **複数名称の併記**: 一般的に「砂揚場」と書かれることも多いため、SEO観点で「旧表記・別名」として本文中に一度含めるのは許容されるが、見出しや強調箇所は必ず**優先表記**を使用すること。
