# 浜名湖フィッシング・リサーチガイド (NotebookLM連携用)

## 🎯 目的
浜名湖における季節・魚種・ポイントごとの生態やフィールド特有のコンテキスト（水温、ベイトパターン、潮流など）を **NotebookLM を使って効率的に収集・整理** し、まとめ記事やポイント解説記事の執筆に必要な高密度なデータ・セット（ソース）を構築する。

これにより、「なぜその時期に、その魚が、そこで釣れるのか」という説得力のある、他サイトと圧倒的に差別化された盤石な記事を作成可能にする。

---

## 📊 NotebookLM にソースとして投入すべき必要要素（データソース）
NotebookLMの「専門家」としての回答精度を高めるためには、あらかじめ以下の質の高いソースファイルを収集し、プロジェクトへ追加（アップロード）しておく必要がある。

1. **季節・水温・気象データ**
   *   例：月別の平均水温、気温、風向き（遠州のからっ風、春一番など）。
   *   有用なソース元：気象庁の過去データ、静岡県水産技術研究所の浜名湖観測データ。
2. **生物・ベイトパターン（食物連鎖）**
   *   例：春のハク（ボラの稚魚）、初夏のクルマエビ・カニ、秋のコノシロ・サヨリ・イワシなど。
   *   各魚種の産卵期（乗っ込み・落ち）の時期とその前後における捕食パターンの変化。
3. **地形・潮流（タイド）データ**
   *   例：浜名湖特有の潮汐（今切口からの潮の入り方とその遅れ）、潮止まりの挙動。
   *   海底地形図、ブレイクライン、アマモ帯の分布、牡蠣棚の位置など。
4. **対象魚の生態方程式（アクティブ条件）**
   *   例：「水温何℃以上で接岸するのか」「潮位の変動（上げ・下げ）のどちらでよく動くか」「夜行性か昼行性か」。
5. **ローカルな実績データと最新トピックス**
   *   例：過去の新聞記事の釣り欄、釣り船の毎日の釣果記録ブログ、地元の釣具店の釣果レポート（PDF化して投入）。

## 📝 リサーチ専用プロンプト集（NotebookLM用）

NotebookLMから効果的に情報を引き出し、ブログ記事の直接の素材となる「最強の中間データ（data-set）」を出力させるためのプロンプトセット。

> **💡 NotebookLMの使い方：**
> 1. 上記の必要要素となるPDFやテキストをNotebookLMにアップロードする。
> 2. 以下のプロンプトをチャット欄に入力し、専門的な回答を生成させる。
> 3. 生成された回答を `data-set/`（知識フォルダ） にMarkdownとして保存し、記事執筆時にAIエージェントにプロンプトとして渡す。

### 🔍 プロンプト1: 特定魚種 × 季節 の行動パターン分析
**用途:** 「〇月のクロダイ」「秋のシーバス」など、特定魚種でのまとめ記事を書くための基礎データ収集。
```text
【役割】
浜名湖の生態系と釣りに精通した海洋生物学者およびプロアングラーとして回答してください。

【対象】
魚種: ◯◯◯◯（例: クロダイ）
季節: ◯◯◯◯（例: 春（3月〜5月））

【指示】
アップロードされたソースドキュメントに基づき、上記の指定された魚種・季節における浜名湖での行動パターンを、以下の項目で詳細に出力してください。ブログ記事の核となる専門的な情報への変換が目的です。

1. 現在の生態ステータス（乗っ込み期、産卵後、越冬前など）と活性
2. 適水温と、この時期の魚の居場所（浅場、深場、アマモ帯、障害物周りなど）の推移
3. メインベイト（何を捕食しているか）と、そのベイトが集まる条件
4. この時期にその魚を釣るために、最も有効なアプローチ（ルアーの種類、エサの種類、時間帯）
```

### 🔍 プロンプト2: 季節 × スタイル による最適ポイントの抽出と理由付け
**用途:** 「夏のファミリー向けサビキ釣りまとめ」など、ターゲットユーザーに合わせたまとめ記事を書くためのデータ収集。
```text
【役割】
浜名湖のすべてのポイントに精通し、初心者から玄人までをガイドするローカルガイドとして回答してください。

【対象】
季節: ◯◯◯◯（例: 夏（6月〜8月））
スタイル: ◯◯◯◯（例: ファミリー向け・足場が良いサビキ釣り）

【指示】
ソースに基づいて、指定された季節とスタイルに最も適した浜名湖の釣りポイントを抽出し、以下のフォーマットでリストアップしてください。ただ名前を挙げるだけでなく「なぜ最適なのか」の明確な理由付けを求めます。

- ポイント名:
- 最適な理由（水温、潮流、回遊ルートなどの環境的要因）:
- 設備のメリット（足場、トイレ、駐車場からの距離など安全面）:
- 狙える代表的な魚と、この時期特有の注意点（熱中症対策、混雑具合、風よけなど）:
```

### 🔍 プロンプト3: 特定ポイントの「地形・潮流」徹底深掘り
**用途:** 既存のポイント記事を大幅にリライト（加筆修正）し、他のブログにはない専門的でマニアックな情報を追加したい時。
```text
【役割】
そのポイントを一途に通い詰めるベテランアングラー（地形マニア）として回答してください。

【対象】
ポイント名: ◯◯◯◯（例: 網干場（舞阪港））

【指示】
ソースに基づいて、このポイントの環境や特徴を以下の観点で詳細に言語化してください。ブログ記事を他の競合サイトと差別化するために、そのまま使える厚みのあるテキストを求めます。

1. 地形の隠れた特徴（シモリ、ブレイク、ミオ筋、潮の抜け方、底質など）
2. 春・夏・秋・冬、それぞれの「このポイントが爆発する（釣れる）瞬間や条件（風向き、潮回りなど）」
3. このポイント特有のローカルルール、マナー、またはアングラー間での暗黙の了解事項
```

## 🔄 AIエージェント（自分）へのデータ受け渡しフロー
NotebookLMで抽出したこれらのデータは、まとめ記事の**「導入部（季節・環境の解説）」** や、ポイント紹介の**「なぜここで釣れるのか（説得力）」** の部分で大いに活用できる。

1. 作成したリサーチ結果を、適当なMarkdownファイル（例：`data-set/spring-kurodai-biology.md`）として保存する。
2. AIにまとめ記事の執筆を依頼する際、`@[data-set/spring-kurodai-biology.md] の情報を参考にして、春のクロダイおすすめポイント記事を作成して` とプロンプトに含めることで、文章の説得力が飛躍的に向上する。
