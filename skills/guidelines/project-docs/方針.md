# 浜名湖釣りガイド 構築方針

ウェブサイトの構成を最小限に構築しなおして、必要な記事を取捨選択する。
ページごとのアクセス数を整理し、WP時代の資産（SEO/URL）を継承しつつ最適化する。

## 🎯 現在の進捗と達成事項

### 魚種別クラスター（Hub-and-Spoke）の確立
主要魚種の標準化を完了。以下の魚種について、Hub記事およびSpoke記事（入門/攻略/ポイント/食）の構成を統一した。
*   **完了済みモデル魚種**: カレイ、ハゼ、メバル・カサゴ
*   **高需要・優先魚種（完了）**: クロダイ、シーバス、カワハギ、キビレ、キス

*   **ビジュアル**: AI生成による一貫性のあるアイキャッチ（16:9）を全記事に配置。
*   **データ構造**: `fishinginfo` フロントマターによる難易度・シーズン・釣り方の正規化。
*   **UI記法**: Markdown標準のアラート記法（`[!TIP]` 等）の全面採用。

## 🚀 今後の方針（Next Steps）

### 1. 残りの魚種および釣法別クラスター化
未完了の魚種についても、同様の Hub-and-Spoke モデルで展開する。
*   **優先（ファミリー・季節需要）**: アジ・サバ（サビキ釣り）、イカ（エギング）
*   **中・長期**: ヒラメ・マゴチ（ルアー・泳がせ）、メジナ、タコ

### 2. 真の「役立つアフィリエイト」の実装
単なるリンクの羅列ではなく、記事の文脈（コンテキスト）に最適化された道具提案を行う。
*   **Beginner（入門）記事**: 
    *   「これさえ買えば間違いない」という、浜名湖のフィールドに特化した **スターターセット** の提案。
    *   ライフジャケットやフィッシュグリップなど、安全・快適に必要な周辺ツールの解説。
*   **Tactics（攻略）記事**: 
    *   特定の釣り方（フリーリグ、ハゼクラ等）で「実際に釣果が出た」具体的な **ルアー、フック、シンカー** の提案。
    *   専門性が高いPEラインやリーダーの推奨設定。
*   **Points（ポイント）記事**:
    *   ポイントの足場状況に合わせた、推奨される **靴（フェルトスパイク等）** や、**水汲みバケツ** の提案。

> [!IMPORTANT]
> **アフィリエイトの実装戦略**: 
> 記事内に直接Amazonリンクを貼るのではなく、商品画像・価格・特徴を美しく表示する `<ProductCard />` コンポーネントを開発する。これにより、サイトのデザインを損なわずに高いクリック率と成約率を目指す。

### 3. WP資産の完全移行とリダイレクト
*   **旧URLの完全継承**: `wpSlug` を活用した 301 リダイレクト設定の最終確認。
*   **月別ポイント記事**: 全月（1~12月）の記事更新とアイキャッチ作成の完了を反映。
