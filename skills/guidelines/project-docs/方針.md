# 今後の方針
ウェブサイトの構成を最小限に構築しなおして、必要な記事を取捨選択する。
ページごとのアクセス数を整理。WPのURLを取得。

## WPで残す記事をリストアップ

### 1~12月のおすすめポイント記事
1~12月のおすすめ記事はすでに作成しておりアクセスもある。
やるべきことは、紹介している記事のlocateを既存のポイント記事から取得。MAP表示してわかりやすいようにする。
リンク先はlist表示。
