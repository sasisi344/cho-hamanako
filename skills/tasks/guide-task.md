# ガイド記事作成タスクリスト

浜名湖釣りの「一次情報」と「正しいデータ」を提供するためのガイド記事拡充計画です。各カテゴリー最低3記事以上の構成を目指します。

## 1. 初心者・入門者向け (`beginner`)
- [ ] **浜名湖釣りのルールとマナー完全ガイド**
  - 内容：釣り禁止エリア、ゴミ、ライフジャケット、地元の規約。
- [ ] **手ぶらで楽しむ！レンタル竿がある釣り施設まとめ**
  - 内容：海釣り公園、ボート店、釣り堀などの実踏データ。
- [ ] **浜名湖の「毒魚・キケン魚」見分け方帳**
  - 内容：ゴンズイ、アイゴ、アカエイ等の対処法。
- [x] ファミリー・車横付けポイント紹介（作成済み）
- [x] サビキ釣りのベストシーズン解説（作成済み）

## 2. インフラ・準備向け (`logistics`)
- [/] **【一次情報】浜名湖の駐車場・トイレ完備ポイントマップ** (下書き作成済み)
- [/] **24時間営業の釣具店＆餌の自販機リスト** (下書き作成済み)
- [/] **浜名湖レンタルボート徹底比較ガイド** (下書き作成済み)
- [x] 釣具店・ショップリスト（基本データ作成済み）

## 3. 釣り方・メソッド向け (`method`)
- [ ] **浜名湖クロダイ・キビレの「トップ」攻略**
  - 内容：聖地浜名湖でのルアー選択と現場のコツ。
- [ ] **奥浜名湖のハゼ釣り完全攻略**
  - 内容：数釣りのための仕掛けと、澪筋（みおすじ）の狙い方。
- [ ] **激流・今切口を攻略するパワーフィッシング術**
  - 内容：重ルアーの扱いと安全な立ち回り。
- [x] 穴釣り入門（作成済み）
- [x] 秋のボート釣り（作成済み）
- [x] 投釣り・キャスティング五目（作成済み）
- [x] 夜釣り入門（作成済み）
- [x] 冬のライトゲーム（作成済み）

## 4. 攻略理論・エキスパート向け (`theory`)
- [ ] **浜名湖「バチ抜け」相関理論**
  - 内容：時期・潮位・エリアごとのバチ抜け発生パターン。
- [ ] **地形を読み解く：澪筋（みおすじ）とカレントの関係**
  - 内容：水の通り道と魚の溜まる場所の図解。
- [ ] **風向きとベイト相関：西風・東風によるポイント選択**
  - 内容：風を利用したポイントの優先順位。
- [x] 浜名湖攻略データベース：水温・潮位ラグ・塩分濃度（作成済み）

- [ ] **【データ検証】全ポイントの駐車場料金・情報の再確認**
  - 背景：新居弁天の料金が400円であるのに対し、AI下書き時に誤情報（500円）が混入したため、データセットやポイント記事内の不整合をチェックする。

---
### 運用ルール
- 記事作成は各ディレクトリの `_draft`（または同等の下書き場所）から開始する。
- 現場の一次情報を重視し、AI生成のみでは到達できない情報を盛り込む。
- 必要に応じて `StreamlitMapButton` を配置し、動的マップと連携させる。
