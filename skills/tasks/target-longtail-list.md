# 🎯 ターゲット別ロングテール・コンテンツ拡充プラン

## 📈 戦略概要：ニッチKWによる流入最大化
「浜名湖 釣り」というビッグキーワードだけでなく、**「浜名湖 × 魚種 × 季節（月）× 釣法」**を組み合わせたロングテールキーワードをターゲットにします。
これにより、特定の悩みやニーズを持つユーザーを確実にキャッチし、最適な釣具やプランの提案（アフィリエイト・旅行予約）へ繋げます。

## 🎣 2. 【釣法×タックル】深掘りコンテンツ
各ロングテール記事から、以下の専門タックル紹介へ誘導し、収益性を高めます。

1.  **「チヌトップ専用」プラグ厳選5選** (6-8月)
2.  **「ハゼクラ」で釣果を伸ばすためのロッド選び** (7-9月)
3.  **「バチ抜けシーバス」攻略のためのシンキングペンシル** (2-4月)
4.  **「投げカレイ」専用、匂いで誘う仕掛けとエサの付け方** (11-1月)
5.  **「浜名湖サビキ」効率重視のセット比較** (7-9月)

## 🛠️ 運用アクションプラン
1.  **カレンダー連動**: 現在の月（3月）に合わせた「バチ抜け」「サヨリ」「乗っ込み」の記事を最優先で作成・補完する。
2.  **内部リンク構造**: `season/`（カレンダー）から各ロングテール記事へ、さらに `target/`（ハブ）へと網目状にリンクを構築する。
3.  **トレンド追従**: 毎週の釣果速報（SNS/地域釣具店）をチェックし、特定の魚種が爆釣している場合は優先順位を上げ、タイトルに「【速報】今釣れてる！」等のフレーズを追加する。
