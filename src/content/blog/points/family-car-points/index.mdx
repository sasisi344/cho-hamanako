## 1. 新居弁天海釣公園：安全なT字堤防と充実の施設

浜名湖で最も有名なファミリー向けポイントといえばここです。

5本の大きなT字型堤防が突き出ており、すべて柵（フェルト）はないものの、足場が非常に広く安定しています。

<Map lat={34.682114} lng={137.589907} name="新居弁天海釣公園" />

<GMapButton url="https://maps.app.goo.gl/9qW8g8YfT6U2" />

*   **おすすめ理由** : T字堤のおかげで潮が緩む場所があり、サビキ釣りに最適です。
*   **施設** : トイレ完備、売店あり、広大な駐車場（400円/回）があります。
*   **狙える魚** : アジ、サバ、イワシ、サヨリ、チンタ（クロダイの幼魚）などが狙えます。

<BlogCard slug="points/omote/araibenten-umiduripark" />

## 2. 弁天島海浜公園：絶景の赤鳥居と駅近の利便性

JR 弁天島駅から徒歩ですぐという、電車釣行も可能な超好アクセスポイントです。

目の前には浜名湖のシンボル ** 「赤い鳥居」 ** がそびえ立ち、ロケーションは最高です。

<Map lat={34.68898} lng={137.60266} name="弁天島海浜公園" />

<GMapButton url="https://maps.app.goo.gl/4fG3Y9J5S8N2" />

*   **おすすめ理由** : 公園として整備されているため、ベンチもあり、散歩感覚で釣りが楽しめます。
*   **施設** : 綺麗なトイレ、コンビニ（ファミマ）至近、広い駐車場が完備されています。
*   **狙える魚** : チンタ、メバル、カサゴ、サヨリなどがターゲットです。

<BlogCard slug="points/omote/bentenjimakaihinkouen" />

## 3. 雄踏山崎：真の「車横付け」フィッシングスポット

「とにかく歩きたくない！」「車のすぐ横で竿を出したい」という方にはこちらが最強です。

道路脇の護岸がそのまま釣り場になっており、まさに車を横付けして釣り座を構えられます。

<Map lat={34.713289} lng={137.618941} name="雄踏山崎" />

<GMapButton url="https://maps.app.goo.gl/T4Z6P8Q9J7L2" />

*   **おすすめ理由** : 車内から竿先を眺める「ズボラ釣り」も可能です。
*   **施設** : ローソンが徒歩圏内にあります。
*   **狙える魚** : 秋のハゼ、キビレ、クロダイなどがよく釣れます。

<BlogCard slug="points/naka/yuto-yamazaki" />

## 4. 気賀（プリンス岬付近）：穏やかな波とハゼ釣りの聖地

奥浜名湖エリアにある気賀は、波が非常に穏やかで湖面が鏡のように美しい場所です。

ハゼ釣りの超有名ポイントで、夏から秋にかけてはのべ竿一本で数釣りが楽しめます。

<Map lat={34.799207} lng={137.623744} name="気賀（プリンス岬）" />

<GMapButton url="https://maps.app.goo.gl/K8Y2X7Q5J4M2" />

*   **おすすめ理由** : 水深が浅く、魚を視認しながら釣れることもあります。
*   **施設** : 駐車スペースがあり、近くにセブンイレブンもあります。
*   **狙える魚** : ハゼ、セイゴ、チンタなどがメインです。

<BlogCard slug="points/oku/kiga" />

## 5. 乙女園：静かに楽しみたい穴場のファミリーポイント

弁天島の西側に位置する、公園併設の静かなポイントです。

海側とは違い、穏やかな入り江になっているため、のんびりと糸を垂らしたい家族におすすめです。

<Map lat={34.69490479372826} lng={137.59440889631668} name="乙女園" />

<GMapButton url="https://maps.app.goo.gl/M4F3P9K2J7H2" />

*   **おすすめ理由** : 芝生広場とトイレがあり、子供が飽きても遊ばせることができます。
*   **施設** : 無料駐車場と公衆トイレがあります。
*   **狙える魚** : ハゼ、キビレ、クロダイなどが狙えます。
