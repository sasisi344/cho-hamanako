## 🔍 ポイント概要と詳細アクセス

乙女園は、弁天島の西側、観月園のさらに奥に位置する静かな公園です。

### 駐車場・アクセス：のんびり派に嬉しい無料駐車場
- <strong>駐車場</strong>: 公園専用の無料駐車場（約20台）があります。
- <strong>公共交通機関</strong>: JR弁天島駅から徒歩約20分。レンタサイクルでのアクセスも快適です。
- <strong>トイレ</strong>: 公園内に完備。管理されており、安心して利用できます。
- <strong>周辺の歴史</strong>: 1923年に建立された「うなぎ観音（魚籃観音）」は、浜名湖のうなぎ供養と湖上の安全を願うシンボルです。

## 🌊 水中地形と「乙女園周辺」の攻略

乙女園の目の前に広がる水域は、比較的浅く穏やかです。

### ① 護岸沿いのシャロー
足元から数メートルは非常に浅く、底が見えることもあります。
- **特徴**: 夏場はハゼの絶好の住処となります。
- **攻略**: 長めののべ竿（3.6〜4.5m）を使い、少し先の底をミャク釣りで探ると、小気味よいハゼのアタリを楽しめます。

### ② 航路側の深み
少し沖には古い航路があり、そこだけ水深が一段深くなっています。
- **解決方法**: チョイ投げで20〜30メートルほど遠投し、カケアガリ（深くなっている斜面）に仕掛けを置くのがクロダイやシロギス攻略の鉄則です。

> [!TIP]
> <strong>木陰でひと休み</strong>
> 乙女園は水辺に木々が多く、夏場でも木陰で涼みながら釣りができる貴重なポイントです。レジャーシートを広げてピクニック気分で楽しむのも良いでしょう。

## 🐟️ 乙女園で狙いたい！四季のターゲット

観音様に見守られながら、季節の魚と対話しましょう。

### 春（4月〜6月）：夜釣りのキビレ＆メバル
日が落ちてからの電気ウキ釣りが面白い時期です。
- <strong>戦略</strong>: 青イソメをエサにし、潮の動きを読みながらゆっくり流します。
- <strong>関連記事</strong>: <a href="/blog/kurodai-beginner">クロダイ初心者ガイド</a>

### 夏（7月〜9月）：ファミリーの主役・ハゼ釣り
乙女園を最も賑わせるのがハゼ釣りです。
- <strong>戦略</strong>: 市頒のハゼ仕掛けにゴールド（ジャリメ）を付けて、底を小刻みに動かします。プルプルという感触はお子様も大喜び間違いなし。
- <strong>関連記事</strong>: <a href="/blog/haze-beginner">ハゼ釣り初心者ガイド</a>

### 秋（10月〜11月）：落ちハゼとクロダイ（チンタ）
ハゼがサイズアップし、クロダイの幼魚「チンタ」も混じるようになります。
- <strong>戦略</strong>: チョイ投げで少し沖を狙います。シロギスが混じることもある嬉しい季節です。
- <strong>関連記事</strong>: <a href="/blog/haze-beginner">ハゼ釣り初心者ガイド</a> / <a href="/blog/kisu-beginner">シロギス初心者ガイド</a>

<a href="/points/fukabori/kisu-fukabori">乙女園・弁天島周辺の「砂地」と「海藻帯」を釣り分けるシロギス攻略。居着きの良型を狙うコツ。</a>

### 冬（12月〜2月）：根魚ライトゲーム
冬の夜、静かな護岸でカサゴやメバルと遊びます。
- <strong>戦略</strong>: 1〜2gのジグヘッドに小型ワームをセット。護岸の際をタイト（ギリギリ）に狙いましょう。
- <strong>関連記事</strong>: <a href="/blog/mebaru-kasago-beginner">メバル・カサゴ初心者ガイド</a> / <a href="/blog/karei-beginner">カレイ初心者ガイド</a>

<a href="/points/kibire-bottom-fukabori">乙女園から望む「ミオ筋（航路）」に潜む大型キビレ。激流のなかで重めのリグを安定させ、ボトムで口を使わせる専門ガイド。</a>

## ⚠️ 公園利用のルールとマナー

皆が気持ちよく過ごせるよう、以下の点にご協力ください。

1. <strong>散歩中の方への配慮</strong>: 後ろを通る歩行者に針を引っ掛けないよう、キャスト時は必ず後方確認を。
2. <strong>ゴミ・エサの清掃</strong>: 釣り座に残ったアミエビやエサは、水で流して綺麗にしましょう。
3. <strong>夜間の静粛</strong>: 近隣は静かなエリアです。夜釣りの際は大きな声での会話は控えましょう。

<Callout type="danger" title="浜名湖の危険生物：アカエイ">
乙女園周辺の浅瀬で立ち込み（ウェーディング）をする際は <strong>アカエイ</strong> に厳重注意してください。移動時は必ず <strong>「すり足（エイガード歩行）」</strong> を行い、砂底のエイを追い散らしながら進みましょう。
</Callout>

<Callout type="important" title="駐車場・清掃マナー">
乙女園公園駐車場（約20台）は <strong>無料</strong> ですが、台数が限られています。路上駐車は観光客や近隣住民の多大な迷惑となるため絶対に厳禁。清掃の行き届いた公園を維持するため、ゴミは必ず持ち帰りましょう。
</Callout>

<Callout type="info" title="浜名湖の歴史に浸る">
釣りの合間に、ぜひ「うなぎ観音」にお参りしてみてください。浜名湖の豊かな恵みへの感謝を込めて手を合わせれば、何かが起こるかもしれません。
</Callout>
